# Data

Its assumed that the following data files exist in `2020_emnlp_curiosity/data`:

- `agreement-krippendorff.pkl`
- `annotated-dataset-MMDD.json`
- `dialog_dataset.json`
- `wiki_sql.sqlite.db`
